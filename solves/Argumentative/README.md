# Argumentative

# Description

<p>Tutorial Challenge. <br/>
Print all of the arguments given to your app (but not your app!)</p>

## Sample Input:

```
$ ./your_app.pl one two three four
```
## Expected Output:

```
$ ./your_app.pl one two three four
one two three four
```
